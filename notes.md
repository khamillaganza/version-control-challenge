# Git and GitHub Notes

## What is Git?
Git is a version control system used to track changes in files and projects.

## What is GitHub?
GitHub is an online platform where developers store and share Git repositories.

## Important Git Commands

### Clone a repository
```bash
git clone repository-url
```

### Check project status
```bash
git status
```

### Add files
```bash
git add .
```

### Create a commit
```bash
git commit -m "message"
```

### Push changes
```bash
git push origin main
```

## Why Version Control Matters
- Tracks changes
- Makes collaboration easier
- Helps recover old versions
- Organizes project history
